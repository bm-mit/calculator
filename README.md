# Bai tap ung dung Calculator
Doan Ngoc Binh Minh


## Setup
- Cai dat NodeJS va Yarn (hoac NPM).

```bash
# cd vao thu muc nay

# install dependencies
yarn install

# run server
yarn dev

```

- Mo trinh duyet va truy cap localhost:3000

## Diem noi bat
- Ho tro nhieu phap tinh toan phuc tap.
- Ho tro dark mode.
